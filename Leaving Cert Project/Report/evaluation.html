Evaluation


Heuristic Evaluation - Nielsen Heuristics


1. Visibility of System Status

The game provides feedback on whether or not the user gives the correct answer. If the user answers incorrectly, the game creates a graph and informs the user that their answer was incorrect.

2. Match Between System and the Real World

The game uses a small sprite of Donald Trump as the player to bring a degree of familiarity to the table. Following convention, the country choices are oriented along the x and y axis.

3. User Control and Freedom

The user, using the WASD or arrow key control scheme, has full freedom of movement in two dimensions. The user has an infinite number of chances to give the correct answer.

4. Consistency of Standard

The user can use the WASD control scheme to control the player. This is standard in the gaming industry.

5. Error Prevention

Error-prone conditions in the game were removed by limiting the area the user could move around. This was achieved using x and y coordinate checks on the user's position.

6. Recognition Rather Than Recall

The framework of the quiz is the same for each question. If the user can answer one question, they will have learned how to answer them all. Instructions on how to navigate around the website are clear and concise.

7. Flexibility and Efficiency of Use

The minimalistic design caters to both experienced and inexperienced users. While there are no accelerators due to time constraints, the design is easy to understand and intuitive to navigate.

8. Aesthetic and Minimalist Design

The design is simple - it consists of a question, four possible answers and a player. It is pleasing to the eye, and the use of the Garamond font in the heading of the page brings an air of tranquility to the page.



Review of the Artefact

Overall, the artefact proved to be an overwhelming success. The artefact successfully fulfilled the conditions specified by the brief, and added an original and interactive spin to the traditional project.
